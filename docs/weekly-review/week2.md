**遇到的问题：**

**我的理解：**
- messages[0] 取的是第一条消息，通常是 system
- messages[-MAX_HISTORY:] 取的是最后几条消息，也就是最近的历史
- [messages[0]] + messages[1:][-MAX_HISTORY:] 的作用是：保留唯一的 system + 保留最近几条非 
- 格式要求不够稳定时，不是模型完全不懂，而是提示词的约束条件还不够明确。
