# AI Agent 学习计划

## 总目标

从 AI Agent 初学者逐步成长为能够独立完成：

- LLM 基础调用
- Prompt 设计
- Tool Calling
- RAG 应用
- ReAct Agent
- Java 工程化落地

并最终沉淀出可展示的项目作品。

---

## 阶段划分

### 第一阶段：LLM 基础入门
目标：
- 理解聊天模型 API 的基本调用方式
- 理解 `system`、`user`、`assistant` 消息结构
- 学会通过 Prompt 完成总结、分类、抽取、改写

输出：
- Hello LLM demo
- Prompt 对比 demo
- 文本任务 demo

---

### 第二阶段：Tool Calling 与 Agent 基础
目标：
- 理解工具调用的原理
- 学会让模型决定是否调用工具
- 手写一个简单 Agent 循环

输出：
- Tool Calling demo
- Mini ReAct Agent

---

### 第三阶段：RAG 应用
目标：
- 理解 embedding、向量检索、文档切分
- 构建个人知识库问答系统

输出：
- 本地知识库 demo
- 带检索的问答应用

---

### 第四阶段：Java 工程化
目标：
- 学习 Spring AI / LangChain4j
- 用 Java 封装 AI 能力
- 面向企业级应用进行架构设计

输出：
- Java Agent demo
- 企业级知识问答系统
- 可扩展的 Agent 服务

---

## 当前执行原则

1. 每天哪怕只学习 15 分钟，也算完成
2. 每周至少保留一个可运行 demo
3. 每周输出一次复盘
4. 学过的 Prompt 和代码要沉淀下来
5. 先完成，再优化，不追求一步到位
